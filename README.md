# Análise do Ranking de Linguagens e Tecnologias

Este repositório contém um projeto completo em Python focado na análise estatística e visualização de dados sobre a popularidade de linguagens de programação, marcação e estilização.

---

## Tecnologias e Bibliotecas Utilizadas

* **Python 3** (Linguagem principal)
* **Pandas** (Manipulação e estruturação de tabelas)
* **Matplotlib** (Plotagem de gráficos de linha e pizza)
* **Jupyter Notebook** (Ambiente interativo de execução)

---


 Gráficos Desenvolvidos

1. **Gráfico de Linha do Ranking:** Exibe a classificação decrescente do mercado, destacando o 1º lugar (Python) no topo da visualização.
2. **Gráfico de Pizza do TOP 5:** Ilustra a representatividade percentual das 5 tecnologias dominantes com efeito de destaque ("explosão").
